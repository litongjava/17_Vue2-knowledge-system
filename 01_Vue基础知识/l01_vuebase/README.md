### 安装
1. 安装Vue开发者工具vue devtools
    1. 只用外网，在谷歌应用商店下载安装 -> 搜索vue devtools -> 点击安装
    2. 自己下载安装：
2. 安装Vue
    参考地址：https://cli.vuejs.org/zh/
    1. 安装`npm install -g @vue/cli`
    2. 创建项目`vue create my-project`  工程名称中不要存在大写字母
3. 运行项目
    在项目的根目录下运行：`npm run serve`
